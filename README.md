## **How to Run**
1. Clone repository ini</br>
```bash
    git clone https://github.com/xmriz/IF2212_TB_08.git 
```

2. Masuk ke folder src</br>
```bash
    cd IF2212_TB_08/src
```

3. compile dan run Main.java</br>
```bash
    javac Main.java
    java Main
```
